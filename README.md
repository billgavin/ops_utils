# PyMail #

**PyMail** is a Python module for working smtplib and Jinja2.

[![Build Status](https://travis-ci.org/billgavin/pymail.svg?branch=master)](https://travis-ci.org/billgavin/pymail)

## Requirements
* Jinja2 >= 2.8

## Docunment ##
#### Getting Started

Install PyMail using pip:

```bash
$ pip install pymail
```
